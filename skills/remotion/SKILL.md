# Remotion — React Video Creation

> Create videos programmatically using React and TypeScript. Render to MP4/WebM.

## Overview
Remotion lets you write videos as React components. Each frame is a React render, giving you full access to the React ecosystem for creating product demos, social clips, and brand videos.

## Prerequisites
- Node.js 18+
- FFmpeg (`brew install ffmpeg`)

## Quick Start
```bash
npx create-video@latest my-video
cd my-video
npm start        # Preview in browser
npx remotion render src/index.ts MyComp out.mp4
```
