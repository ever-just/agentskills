# Templates

## Level 1: SKILL.md (Minimal)

```markdown
---
name: skill-name
description: Does X when Y. Use for Z.
---
# Skill Name

Brief one-liner.

## Resources
- [Guide](docs/guide.md)
```

## Level 2: Simple Skill

```markdown
# Skill Guide

## Quick Start
[Essential steps]

## Details
[Core instructions]

## Examples
[Concrete examples]
```

## Level 2: Index (Complex Skill)

```markdown
# Skill Index

## Topics
- [Topic A](topic-a.md) - brief description
- [Topic B](topic-b.md) - brief description
- [Topic C](topic-c.md) - brief description

## Quick Reference
[Most common case inline]
```

## Level 3: Topic File

```markdown
# Topic A

## Overview
[1-2 sentences]

## Instructions
[Step-by-step]

## Examples
[Input/output pairs]
```

## Description Formula

```
[Verb phrase: what it does]. Use when [trigger conditions].
```

Examples:
- `Extract text from PDFs. Use when processing PDF files or documents.`
- `Generate commit messages from diffs. Use when committing changes.`
- `Review code for quality issues. Use when reviewing PRs or code.`
