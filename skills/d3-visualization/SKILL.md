# D3.js — Data Visualization

> Create animated, data-driven charts, graphs, and interactive visualizations.

## Overview
D3.js binds data to DOM elements and applies data-driven transformations. Ideal for custom charts, maps, and animated data stories.

## Prerequisites
- Node.js 18+

## Quick Start
```bash
npm install d3
```
