# Remotion Captions — Animated Subtitles

> Add animated subtitles and captions to Remotion videos from SRT files.

## Overview
Renders SRT/VTT caption files as animated, styled overlays in Remotion videos with word-level timing.

## Prerequisites
- Node.js 18+
- Remotion installed
