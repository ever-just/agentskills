# Motion Canvas — TypeScript Animation Engine

> Create explainer videos and code walkthroughs with TypeScript.

## Overview
Motion Canvas is a TypeScript library for creating animated explainer videos using a generator-based API for precise timing control.

## Prerequisites
- Node.js 18+

## Quick Start
```bash
npm create @motion-canvas@latest
cd my-animation
npm start
```
