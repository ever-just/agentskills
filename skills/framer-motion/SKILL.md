# Framer Motion — React UI Animations

> React UI animations, page transitions, gesture interactions, and layout animations.

## Overview
Framer Motion (now Motion) provides declarative animations for React with spring physics, gesture support, and layout animations.

## Prerequisites
- Node.js 18+
- React 18+

## Quick Start
```bash
npm install motion
```
