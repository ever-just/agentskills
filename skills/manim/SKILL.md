# Manim — Python Math/Science Animations

> Create beautiful math and science animations with Python.

## Overview
Manim (Mathematical Animation Engine) creates precise mathematical visualizations and algorithm animations.

## Prerequisites
- Python 3.9+
- FFmpeg (`brew install ffmpeg`)

## Quick Start
```bash
pip install manim
manim -pql scene.py MyScene
```
