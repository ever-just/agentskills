# GSAP — Web Animation Library

> Professional-grade web animations, scroll effects, and SVG animations.

## Overview
GSAP (GreenSock Animation Platform) provides high-performance animations for the web with timeline-based sequencing and scroll-linked effects.

## Prerequisites
- Node.js 18+

## Quick Start
```bash
npm install gsap
```
