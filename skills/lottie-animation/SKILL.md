# Lottie Animation — JSON Animations

> Create animated logos, icons, and micro-interactions as Lottie JSON files.

## Overview
Lottie renders After Effects animations exported as JSON. Perfect for lightweight, scalable animations on web and mobile.

## Prerequisites
- Node.js 18+

## Quick Start
```bash
npm install lottie-web
```
