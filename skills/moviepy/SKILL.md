# MoviePy — Python Video Editing

> Programmatic video editing, compositing, and frame-by-frame effects with Python.

## Overview
MoviePy provides a Python interface for video editing — cutting, concatenating, adding effects, compositing, and exporting.

## Prerequisites
- Python 3.9+
- FFmpeg (`brew install ffmpeg`)

## Quick Start
```bash
pip install moviepy
```
